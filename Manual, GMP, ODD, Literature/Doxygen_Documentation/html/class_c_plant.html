◆ Kill()

void CPlant::Kill ( )
virtual

Mortality due to resource shortage or at random.

Calculate weekly individual mortality. We assume an base mortality of 0.007 and add a stress dependent mortality. This is a stochastic process (mortality rate is interpreted as probability here). If the Plant is killed, it gets the status dead=true. Subsequently the litter gets decomposed weekly.
