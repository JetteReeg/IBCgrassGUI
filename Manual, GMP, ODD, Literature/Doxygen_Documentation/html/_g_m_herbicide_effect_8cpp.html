The experimental design is read from program arguments (sample values in brackets): general parameters

  1. ModelVersion (3); // mode of density denpendent mortality and competition
  2. CellNum (173); // grid size (cm)
  3. Tmax (100); // maximal number of years simulated
  4. Tinit (50); // initial years before herbicide effects are simulated
  5. NamePftFile ("Fieldedge.txt"); // PFT list
  6. seedsPerType (10); // degree of isolation as number of seed input per year resources
  7. meanBRes (90); // belowground resources
  8. meanARes (100); // aboveground resources
  9. Aampl (0.7); // amplitude of seasonal distribution of aboveground resources (based on day length)
  10. Bampl (0); // amplitude of seasonal distribution of belowground resources disturbances
  11. AreaEvent (0.1); // amount of area trampled per year
  12. GrazProb (0.01); // amount of area grazed per year
  13. NCut (1); // number of cutting events per year herbicide impact parameter
  14. week_start (1); // week of application of herbicide
  15. Generation ("F0"); // affected generation
  16. HerbDuration (30); // number of years simulated with herbicide effect
  17. HerbEffectType (1); // defines simulation run as treatment of control (1 - treatment; 0 - control)
  18. EffectModel (0); // defines on which input herbicide effects are based (0 - txt file 2 - dose response)
  19. scenario (1); // number of scenario
  20. MCrun (1); // number of monte carlo runs

Output file names are given for each run individually.
